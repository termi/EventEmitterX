# Идеи для видео

## Как я делал определение "Столицы" и "таймзоны" по локале

* Использование библиотек `world-countries-capitals` и `countries-and-timezones`
* Написание `postintall` скрипта, который создаёт собственный JSON для хранения ТОЛЬКО НУЖНОЙ информации
* Т.к. у нас по-умолчанию используется русский язык для авто-перевода, то нужно названия столиц загруженные из этой базы переводить с английского
