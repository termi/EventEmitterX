## Human-Readable Summary

### Demo App — New Files
- **`lib/cache.ts`** — `WeakCache<K, V>` — WeakMap + WeakRef-based cache with `.emplace()` API (insert-or-update)
- **`lib/object.ts`** — `shallowEqual`, `getIterableDiff`, `checkWeakRefSupported`, `checkHasWeekMapSymbolsSupport`
- **`lib/weather.ts`** — WIP: weather data via Open-Meteo API (not yet wired up).
  Add todo for using getWeather in `pages/10.GlobalTimes.tsx` to show current weather in the city info section.
- **`lib/polyfills/symbol.ts`** — Symbol polyfills

### Demo App — Changes
- **`vite.config.ts`**: replaced built-in `vite:json` plugin with a custom JSON loader that emits `JSON.parse(\`...\`)` — avoids embedding large JSON as JS object literals, which is slower to parse per V8 blog post
- **`lib/dom.ts`**: added `getFormValuesAsObject()` and `getFormElementValue()` utilities; added time constants
- **`state/GlobalTimesState.ts`**: `mostPopularCities$` computation now uses `getIterableDiff()` — only destroys removed city signals (previously ALL were destroyed and recreated on locale change); added `WeakCache` to reuse city time signals; added `emptyCitySignal$` lazy getter in `.data` as a fallback
- **`state/i18n.ts`**: `i18nNumber.use()` now passes `shallowEqual` to prevent re-renders when `numberFormatOptions` hasn't changed; added `i18n$$.componentType` and `i18nString$$.componentType`
- **`state/AppStates.ts`**: field titles and placeholders now use `i18n$$` tagged template for i18n; added `inputProps` to `userFirstName$` and `userSecondName$` data
- **`pages/2.two.tsx`**: added `useActionState` hook; form now uses `getFormValuesAsObject` in `onSubmit`; `$mappedSignal` → `mappedSignal$`
- **`pages/10.GlobalTimes.tsx`**: `currentLocaleCityInfo$` now falls back to `emptyCitySignal$`; replaced `useListener` with `useReducedListener` for DOM attribute updates
- **`modules/DeferredSearchResults.tsx`**: removed local `use<T>` workaround, now uses React 19's built-in `use()`
- **`modules/EventSignalSearchResults.tsx`**: `retry` moved from `data` object to direct `current$.retry` method call
- **`modules/TicTacToe/TicTacToeGameMutablePro.tsx`**: removed `React.memo()` wrapper, added `rendersCounter` for debugging
- **`modules/TicTacToe/TicTacToeGameMutable.tsx`**: fixed render counter (pre-increment `++`)
